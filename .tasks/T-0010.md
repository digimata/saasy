# T-0010 :: Audit auth code — prune dead/vendored components

Audit `apps/web/components/auth/`, `apps/web/lib/auth/`, `apps/web/hooks/auth/`, and `packages/auth/` for dead code, unused vendored components, and redundant files. The auth UI was vendored from a library and likely has components we don't use. Identify what's live, what's dead, and prune.

## Scope

- Map every file in auth-related directories
- Cross-reference imports to confirm usage
- Remove dead components and their imports
- Clean up any orphaned types, utils, or hooks
