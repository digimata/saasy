# T-0005 -- Billing stub -- Stripe customer + portal

## Context

The template needs a minimal Stripe integration point so projects cloned from it have billing wiring ready. Not a full billing system — just customer creation and portal access.

## Goal

Create a Stripe customer on workspace setup. Add a billing page with tier/status display and a link to Stripe Customer Portal.

## Criteria

- Stripe customer created when workspace is created
- Billing page shows current tier and subscription status
- Link to Stripe Customer Portal for self-service management
- Stripe webhook endpoint for subscription lifecycle events
