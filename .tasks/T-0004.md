# T-0004 -- Workspace CRUD + settings

## Context

The schema supports workspaces and members but there's no UI for managing them. Need workspace creation on signup and a settings page.

## Goal

Wire workspace creation into the signup flow (implicit 1:1). Build workspace settings page (name, slug). Wire the workspace switcher in the sidebar.

## Criteria

- Default workspace created on signup
- Workspace settings page (edit name, slug)
- Workspace switcher in sidebar shows user's workspaces
- Member list page (read-only for MVP)
