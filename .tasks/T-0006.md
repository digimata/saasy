# T-0006 -- Wallet login + 2FA plugins

## Context

BetterAuth has SIWE and TOTP plugins. These need to be wired into the auth config and UI. Wallet login is important for agent-native onboarding; 2FA is a baseline security feature.

## Goal

Add SIWE (EVM wallet) login flow and TOTP 2FA to the auth config. Build UI for both.

## Criteria

- SIWE plugin configured in BetterAuth
- Wallet connect button on sign-in/sign-up pages
- 2FA (TOTP) plugin configured
- 2FA setup page (QR code, backup codes)
- 2FA verification on login when enabled
