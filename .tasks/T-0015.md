# T-0015: Bot protection on auth forms

Add Cloudflare Turnstile to auth forms (sign-in, sign-up, OTP request) to prevent automated abuse.

## Scope

- Turnstile invisible widget on auth forms (no user friction)
- Server-side token verification before processing auth requests
- Rate limiting on auth endpoints (Upstash Ratelimit) as a second layer
- Security headers in next.config (CSP, HSTS, X-Frame-Options)

## Notes

- Turnstile is free, invisible mode requires no user interaction
- Vercel handles DDoS/TLS/edge but not application-level rate limiting or bot detection on forms
- Upstash Ratelimit has a free tier, ~3 lines in middleware
